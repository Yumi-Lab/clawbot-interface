CLI
Switch AI agent
PicoClaw (<10MB RAM, recommended) vs OpenClaw (~500MB). Use the Services panel or the CLI command:
sudo clawbot-switch-agent picoclaw
sudo clawbot-switch-agent openclaw
sudo clawbot-switch-agent status

How-to
Enable Telegram bot
1. Create bot via @BotFather → copy token 2. Enter token in Settings → Telegram Bot 3. Go to Services → Start clawbot-telegram 4. Message your bot on Telegram — it should reply!
